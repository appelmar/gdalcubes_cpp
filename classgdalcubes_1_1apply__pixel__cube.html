Detailed Description

A data cube that applies one or more arithmetic expressions on band values per pixel.

Note
This class either works either with exprtk or with tinyexpr, depending on whether USE_EXPRTK is defined or not. Please notice that the functionality of these libraries (i.e. the amount of functions they support) may vary. tinyexpr seems to work only with lower case symbols, expressions and band names are automatically converted to lower case then.

Parameters

band_namesspecify names for the bands of the resulting cube, if empty, "band1", "band2", "band3", etc. will be used as names
keep_bandsif true, bands will be added to the existing bands of the input cube, otherwise (default) they are dropped
