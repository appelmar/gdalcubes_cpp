Detailed Description

A data cube view includes the spacetime reference of a cube (extent, resolution, projection) and optional resampling and aggregation algorithms that are applied when original images are read from an image_collection_cube. Aggregation refers to how multiple values for the same cube cell from different images are combined whereas resampling refers to the algorithm used to warp / reproject images to the cube geometry.
